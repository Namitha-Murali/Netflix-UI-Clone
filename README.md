🎬 Netflix Replication
This project showcases a simple yet effective replication of the Netflix interface, created using HTML, CSS, and JavaScript. The goal was to mimic the user experience of the popular streaming platform while maintaining a straightforward and accessible codebase. 🚀

✨ Features
Responsive Design: 📱 The layout adapts seamlessly to various screen sizes, providing an optimal viewing experience on both desktops and mobile devices.
Interactive UI: 🎨 Enjoy smooth navigation through sections reminiscent of Netflix, complete with elegant transitions and captivating hover effects.
Dynamic Content: 🔄 Utilizing JavaScript to simulate content loading and interactivity, this project offers a feel of a real-world application.
💻 Tech Stack
HTML: Structuring the content and layout for clear organization.
CSS: Styling the application for visual appeal and responsive design.
JavaScript: Adding interactive features and dynamic content management.
